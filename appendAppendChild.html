Difference between append and appendChild

append() => you can append an Element (Node or DOMString) to the DOM.

1. Append Element
2. Adding multiple Elements
3. append() does NOT actually return values


Summary =>

No return value
Support multiple Element append
You can add Node & DOMString

appendChild() => Using appendChild, you can only append a Node.

When you try to add "String" to the appendChild(), then it will throw error

Using appendChild, you can only append ONLY ONE Node in single method call


Summary =>

Supports only Node addition
Returns the appended Node
Only one element can be appended
