Submit event listener

How to get a value property?

event.preventDefault()

When we are working with forms,
We have an action="" attribute

What action attribute does,
it tells where we are going to be sending the data

By default, we are grabbing the values and then sending them to server.
so if we wouldn't want the page to refresh by default behavior then we need to use preventDefault().

Why?

By default, because we are not just sending the data to where the action tells us

BUT also we are REFRESHING the page

so it is impossible for us to grab the value if the page keeps refreshing
